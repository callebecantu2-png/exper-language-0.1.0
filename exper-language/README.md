# Exper Language

Uma linguagem de programação criada em Python com interpretador próprio.

## Features
- Variáveis
- Funções (fn)
- if / else / elif
- while / for
- listas e dicionários
- console.log
- input
- strings com interpolação {variavel}

## Exemplo

```exper
nome = console.input("Nome: ")
console.log("Olá {nome}")
